# CRM Clientes

Sistema CRM completo para gerenciamento de clientes, autenticação e controle de permissões.

## Tecnologias

- Django
- Django REST Framework
- React.js
- TypeScript
- PostgreSQL
- Docker

## Funcionalidades

- Cadastro de clientes
- Autenticação JWT
- Controle de permissões
- Dashboard administrativo
- CRUD completo
